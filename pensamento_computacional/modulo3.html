Módulo 3: Introdução ao JavaScript

Neste módulo você aprenderá os conceitos básicos de JavaScript, a linguagem de programação usada para adicionar interatividade a páginas web. É o primeiro passo para criar sites dinâmicos e responsivos.

  • ✔ O que é JavaScript e onde é utilizado
  • ✔ Inserindo scripts em páginas HTML
  • ✔ Variáveis, operadores e tipos de dados
  • ✔ Funções e manipulação de eventos
  • ✔ Introdução ao DOM (Document Object Model)

Aprender JavaScript permite criar experiências interativas em sites e aplicações web. É uma habilidade essencial para qualquer desenvolvedor back-end.
